# Зображення для сайту

## Структура папок

- `hero/` - зображення для hero секції (карусель)
- `products/` - зображення продуктів
- `suppliers/` - логотипи постачальників
- `gallery/` - зображення для галереї

## Як завантажити зображення зі старого сайту

### Варіант 1: Через браузер (найпростіше)

1. Відкрийте https://rvdsever.wixsite.com/website
2. Відкрийте DevTools (F12)
3. Перейдіть на вкладку Network → Img
4. Перезавантажте сторінку (F5)
5. Знайдіть потрібні зображення та завантажте їх:
   - Клікніть правою кнопкою на зображення
   - Виберіть "Open in new tab"
   - Збережіть зображення (Ctrl+S)
   - Перемістіть у відповідну папку

### Варіант 2: Через інспектор елементів

1. Відкрийте https://rvdsever.wixsite.com/website
2. Клікніть правою кнопкою на зображення → "Inspect"
3. Знайдіть URL зображення в атрибуті `src`
4. Відкрийте URL в новій вкладці
5. Збережіть зображення
