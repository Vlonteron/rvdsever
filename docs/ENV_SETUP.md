# Налаштування змінних оточення

## Створення .env.local файлу

Створіть файл `.env.local` в корені проекту з наступним вмістом:

```env
PORT=3000

# Telegram Bot налаштування для форми зворотного зв'язку
TELEGRAM_BOT_TOKEN=your_bot_token_here
TELEGRAM_CHAT_ID=your_chat_id_here
```

## Порт для запуску

Порт для запуску сервера можна налаштувати через змінну оточення `PORT`.

### Використання

Next.js автоматично читає змінні з файлу `.env.local` при запуску.

**Для зміни порту:**
1. Відредагуйте файл `.env.local`
2. Змініть значення `PORT` на потрібне (наприклад, `PORT=3001`)
3. Перезапустіть сервер

**Альтернативно, можна встановити порт через командний рядок:**

Windows (PowerShell):
```powershell
$env:PORT=3001; npm run dev
```

Windows (CMD):
```cmd
set PORT=3001 && npm run dev
```

Linux/Mac:
```bash
PORT=3001 npm run dev
```

## Налаштування Telegram бота для форми зворотного зв'язку

### Крок 1: Створення Telegram бота

1. Відкрийте Telegram і знайдіть бота [@BotFather](https://t.me/BotFather)
2. Надішліть команду `/newbot`
3. Дотримуйтесь інструкцій для створення бота
4. Після створення ви отримаєте **Bot Token** (виглядає як `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### Крок 2: Отримання Chat ID

**Chat ID** — це унікальний ідентифікатор вашого Telegram-акаунта, групи або каналу, куди будуть надсилатися повідомлення з форми.

#### Спосіб 1: Через бота @userinfobot (найпростіший для особистих повідомлень) ⭐

1. Відкрийте Telegram
2. Знайдіть бота [@userinfobot](https://t.me/userinfobot)
3. Натисніть **"Start"** або надішліть будь-яке повідомлення (наприклад, `/start`)
4. Бот одразу надішле вам ваш **Chat ID** — це число, наприклад: `123456789`
5. Скопіюйте це число — це і є ваш `TELEGRAM_CHAT_ID`

**Приклад відповіді бота:**
```
Your user ID: 123456789
Your username: @your_username
```

#### Спосіб 2: Через API Telegram (универсальний метод)

1. Спочатку отримайте **Bot Token** (якщо ще не маєте — див. Крок 1)
2. Відкрийте в браузері наступну URL (замініть `YOUR_BOT_TOKEN` на ваш токен):
   ```
   https://api.telegram.org/botYOUR_BOT_TOKEN/getUpdates
   ```
   **Приклад:** `https://api.telegram.org/bot123456789:ABCdefGHIjklMNOpqrsTUVwxyz/getUpdates`

3. Якщо ви ще не надсилали повідомлення боту:
   - Знайдіть свого бота в Telegram (за ім'ям, яке ви дали при створенні)
   - Натисніть **"Start"** або надішліть будь-яке повідомлення
   - Оновіть сторінку в браузері

4. У відповіді знайдіть поле `"chat":{"id":123456789}` — число після `"id":` і є вашим Chat ID

**Приклад відповіді API:**
```json
{
  "ok": true,
  "result": [{
    "message": {
      "chat": {
        "id": 123456789,  ← Це ваш Chat ID!
        "first_name": "Ваше Ім'я"
      }
    }
  }]
}
```

#### Спосіб 3: Для груп/каналів

Якщо ви хочете отримувати повідомлення в групу або канал:

1. Додайте свого бота в групу/канал як адміністратора
2. Надішліть будь-яке повідомлення в групу/канал
3. Використайте Спосіб 2 (через API) — Chat ID групи/каналу буде від'ємним числом, наприклад: `-1001234567890`

**Примітка:** Для каналів Chat ID завжди починається з `-100`, для груп — з `-`.

#### Спосіб 4: Через бота @getidsbot

1. Знайдіть бота [@getidsbot](https://t.me/getidsbot)
2. Надішліть команду `/start`
3. Отримайте ваш Chat ID

### Крок 3: Додавання змінних у .env.local

Додайте отримані значення в файл `.env.local`:

```env
TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
TELEGRAM_CHAT_ID=123456789
```

### Перевірка налаштування

Після налаштування перезапустіть сервер розробки:

```bash
npm run dev
```

Тепер форма зворотного зв'язку буде автоматично надсилати повідомлення в ваш Telegram!

### Примітка

- Файл `.env.local` вже додано в `.gitignore` і не буде закомічений у репозиторій
- Якщо `PORT` не встановлено, Next.js використовує порт за замовчуванням: `3000`
- Для production сервера також можна використовувати змінну `PORT`
- **НЕ діліться** своїм Bot Token та Chat ID публічно!
