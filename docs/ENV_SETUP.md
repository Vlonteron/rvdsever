# Налаштування змінних оточення

## Порт для запуску

Порт для запуску сервера можна налаштувати через змінну оточення `PORT`.

### Створення .env.local файлу

Створіть файл `.env.local` в корені проекту з наступним вмістом:

```env
PORT=3000
```

### Використання

Next.js автоматично читає змінні з файлу `.env.local` при запуску.

**Для зміни порту:**
1. Відредагуйте файл `.env.local`
2. Змініть значення `PORT` на потрібне (наприклад, `PORT=3001`)
3. Перезапустіть сервер

**Альтернативно, можна встановити порт через командний рядок:**

Windows (PowerShell):
```powershell
$env:PORT=3001; npm run dev
```

Windows (CMD):
```cmd
set PORT=3001 && npm run dev
```

Linux/Mac:
```bash
PORT=3001 npm run dev
```

### Примітка

- Файл `.env.local` вже додано в `.gitignore` і не буде закомічений у репозиторій
- Якщо `PORT` не встановлено, Next.js використовує порт за замовчуванням: `3000`
- Для production сервера також можна використовувати змінну `PORT`
