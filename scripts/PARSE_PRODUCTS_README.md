# Парсинг информации о продукции с Wix сайта

## Использование

Для парсинга информации о продукции с Wix сайта выполните:

```bash
npm run parse-products
```

## Что делает скрипт

1. Загружает страницу продукции: `https://rvdsever.wixsite.com/website/продукція`
2. Пытается найти ссылки на детальные страницы каждого продукта
3. Пытается получить информацию с каждой страницы продукта
4. Сохраняет результаты в `lib/product-details.json`

## Ограничения

⚠️ **Важно**: Wix сайты используют динамическую загрузку контента через JavaScript, поэтому автоматический парсинг может не получить всю информацию.

## Ручное обновление данных

Если автоматический парсинг не сработал, выполните следующие шаги:

1. Откройте страницу продукции в браузере: https://rvdsever.wixsite.com/website/продукція
2. Перейдите на каждую страницу продукта, нажав "Докладніше"
3. Скопируйте информацию с каждой страницы:
   - Описание
   - Детальная информация
   - Характеристики
   - Особенности
   - Изображения
4. Обновите файл `lib/product-details.ts` с полученной информацией

## Структура данных

Каждый продукт должен иметь следующую структуру:

```typescript
{
  id: number;
  title: string;
  description: string;
  details?: string;           // Детальная информация
  images?: string[];          // URL изображений
  specifications?: string[]; // Характеристики
  features?: string[];       // Особенности
}
```

## Результаты

После выполнения скрипта:
- `lib/product-details.json` - JSON файл с результатами парсинга
- `lib/products-page.html` - HTML страницы продукции для ручного анализа
